    Create an Element

    task
    After you create your app, you implement these additional steps to create an element.

    Before you can begin coding your element, make sure you first complete the initial development steps for all apps.

    Then follow these element-specific instructions:

    1. Set up the additional file structure needed for elements. Your template, JavaScript, CSS and other files, must reside is specific directories when you create your app bundle.
    2. Add element information to the manifest. Then create your settings in the manifest.
